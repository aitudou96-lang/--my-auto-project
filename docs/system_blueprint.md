# 系统蓝图

## 总目标

围绕“售卖智能体”搭建一套可持续运转的内容生产系统，而不是只追求泛流量。

## 三类内容模型

- A 类：拉新流量内容
- B 类：建立信任内容
- C 类：转化内容

## 五大内容支柱

- A：AI 小白入门
- B：AI 实操教程
- C：创业者提效与变现
- D：智能体案例与场景
- E：认知升级与避坑

## 每日执行主流程

1. 信息抓取：AI 热点、工具、讨论、竞品、起量样本
2. 筛选评分：按热度、匹配度、转化潜力等打分
3. 爆款拆解：拆标题、钩子、冲突、叙事、CTA、评论关注点
4. 原创重组：至少改变 3 个以上核心维度
5. 每日导出：选题总览、重点脚本包、剪辑辅助包、待办清单
6. 结果回写：更新知识库、标题库、钩子库、失败案例库

## 周复盘输出

- 本周爆款共性
- 本周失败共性
- 本周新增有效标题模板
- 本周新增有效开头模板
- 本周新增高转化内容模型
- 哪类内容适合卖智能体
- 哪类内容只能吸粉
- 下周优先测试方向
- 该淘汰什么内容形式
